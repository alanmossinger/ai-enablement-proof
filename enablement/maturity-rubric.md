# AI Enablement Maturity Rubric

5 levels x 6 dimensions. AI-Enablement-Proof is scored honestly, including shortfalls.

## Dimensions

### 1. Governance

| Level | Description |
|-------|-------------|
| 1 - Ad hoc | No formal AI governance; decisions made case-by-case |
| 2 - Documented | Governance policies exist as documents |
| 3 - Enforced | Governance is coded into CI/CD pipelines |
| 4 - Measured | Governance effectiveness is tracked with metrics |
| 5 - Adaptive | Governance evolves based on measured outcomes |

### 2. Data Ownership

| Level | Description |
|-------|-------------|
| 1 - Ad hoc | No clear data ownership; access is informal |
| 2 - Cataloged | Data assets inventoried, owners assigned |
| 3 - Quality-managed | Data quality gates enforce standards |
| 4 - Governed | Lineage tracked, point-in-time access enforced |
| 5 - Self-service | Business users access governed data independently |

### 3. Workforce Literacy

| Level | Description |
|-------|-------------|
| 1 - Awareness | Leadership knows AI exists |
| 2 - Understanding | Teams understand AI capabilities and limits |
| 3 - Applied | Teams use AI tools in daily workflows |
| 4 - Building | Teams build and evaluate AI solutions |
| 5 - Teaching | Organization exports AI knowledge externally |

### 4. Use-Case Portfolio Discipline

| Level | Description |
|-------|-------------|
| 1 - Ad hoc | Use cases emerge randomly |
| 2 - Intake | Formal intake process exists |
| 3 - Prioritized | Portfolio scored on value/feasibility/risk |
| 4 - Measured | ROI tracked per use case |
| 5 - Strategic | Portfolio drives capital allocation |

### 5. Value Measurement

| Level | Description |
|-------|-------------|
| 1 - Anecdotal | "It seems to help" |
| 2 - Activity-based | Usage metrics (logins, queries) |
| 3 - Outcome-based | Business metrics tied to AI (time saved, errors avoided) |
| 4 - Financial | Dollar value with counterfactual structure |
| 5 - Auditable | Value claims reproducible from committed artifacts |

### 6. Operating Cadence

| Level | Description |
|-------|-------------|
| 1 - None | No regular review |
| 2 - Quarterly | Quarterly steering committee |
| 3 - Monthly | Monthly scorecard reviews |
| 4 - Weekly | Weekly operational reviews |
| 5 - Continuous | Real-time dashboards + escalation triggers |
