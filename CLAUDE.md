## Project Context

AI-Enablement-Proof is a governed autonomous agent that monitors U.S. grid reliability
using public federal data (EIA-930, NOAA/NWS, ERCOT archives). Every claim must be
reproducible from committed code and public data.

## Non-Negotiable Rules

1. **No fabricated data.** Never generate synthetic data and present it as real.
   Every number traces to a public source or committed notebook.

2. **Point-in-time discipline.** Backtests must use only data available at the
   decision moment. Lookahead contamination is a project-invalidating bug.

3. **No implied operational deployment.** Always use "demonstration system,"
   "backtested," "running on live public data." Never imply a grid operator uses this.

4. **No employer names.** Anonymized vignettes only.

5. **Anti-fabrication over speed.** If uncertain whether a claim is reproducible,
   flag it rather than commit it. A slower honest build beats a fast dishonest one.

6. **Governance gates are code, not prose.** `src/governance/gates.py` enforces
   constraints programmatically. `test_no_L5.py` is a required CI check.

7. **No AI attribution in commits.** Do not add Co-Authored-By or other AI mentions
   to commits in this repository.

## Code Standards

- **Python 3.11+**, type hints on public interfaces
- **Testing:** pytest, governance tests are required CI checks
- **Formatting:** ruff (format + lint)
- **Imports:** stdlib > third-party > local, one blank line between groups
- **Docstrings:** only on public API functions; keep them concise
- **Error handling:** validate at system boundaries (API inputs, external data);
  trust internal code
- **Dependencies:** minimal — prefer stdlib; add packages only when clearly needed

## Architecture Principles

- DuckDB for point-in-time data store (as-of query semantics)
- LightGBM baseline before any complex model
- Governance gates are the crown jewels — treat `src/governance/` with extra care
- Audit trail is append-only JSONL — never delete or modify logged events
- Every figure generated by committed code from committed data
